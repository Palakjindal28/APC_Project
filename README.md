# 📚 Online Learning System

An Online Learning System built using Java and Spring Boot that enables users to enroll in courses, manage learning content, and handle authentication securely through a RESTful backend architecture.

---

## 🚀 Features

* 🔐 User Authentication (Login/Signup with secure handling)
* 📘 Course Management (Create, update, delete courses)
* 🎓 Student Enrollment System
* 📊 Efficient data handling using relational databases
* 🔄 RESTful APIs for seamless communication
* 🧩 Layered architecture (Controller → Service → Repository)

---

## 🛠️ Tech Stack

**Backend:** Java, Spring Boot

**Database:** MySQL / PostgreSQL

**Tools:** Postman, Git, Maven

---


## ⚙️ Installation & Setup

1. Clone the repository

```bash id="p04y6f"
git clone https://github.com/Palakjindal28/APC_Project.git
```

2. Navigate to project

```bash id="92fx2l"
cd online-learning-system
```

3. Run the application

```bash id="q7qk2g"
mvn spring-boot:run
```

---

## 🔑 Configuration

Update your `application.properties` file:

```properties id="cx3s5p"
spring.datasource.url=jdbc:mysql://localhost:3306/your_db
spring.datasource.username=root
spring.datasource.password=your_password

spring.jpa.hibernate.ddl-auto=update
```
